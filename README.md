# newsSummary

#### 프로젝트 설명
- 해당 프로젝트는 python beautifulSoup 파싱과 KoBart를 이용한 파싱을 기반으로 데이터를 갱신한다.
<br/>
- 해당 데이터는 React로 작성된 Front에서 rest api를 이용해 Spring boot 웹 서버로 idx값을 포함하여 전송하면
웹 서버는 python에서 얻은 정보를 DB에 검색하고 해당 데이터를 Front로 재 전송해준다.
